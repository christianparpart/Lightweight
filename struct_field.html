Detailed Description

template<detail::FieldElementType T, auto P1 = std::nullopt, auto P2 = std::nullopt>
struct Field< T, P1, P2 >

Represents a single column in a table.

The column name, index, and type are known at compile time. If either name or index are not known at compile time, leave them at their default values, but at least one of them msut be known.

It is imperative that this data structure is an aggregate type, such that it works with C++20 reflection.
